Lightbox CLONE

is small plugin, used to overlay images on top of the current page.

This plugin is an example of how OmniWindow can be easily customized to fulfil the functions of Lightbox library.

Examples

How to use

1. Include jQuery, OmniWindow and plugin libs to your html

  
  <script type="text/javascript" src="...jquery.js"></script>
  <script type="text/javascript" src="...jquery.omniwindow.js"></script>
  <script type="text/javascript" src="...lightbox_clone.js"></script>

2. Add container for the overlay and modal window

  
  <div class="ow-overlay ow-closed"></div>

  <div class="image-modal ow-closed">
    <img class="loader" src='loading.gif' width="32" height="32">
    <p class="modal-caption"></p>
    <span class="close-modal">+</span>
  </div>

You'll need to put close button, caption and loader image within the modal window container.

Don't forget to download and add loading.gif image to your project.

3. Add default css

  
  <link rel="stylesheet" type="text/css" href="...lightbox_clone.css" media="all">

4. Add image gallery structure

The default structure is as follows:

  
  <ul class="image-gallery">
    <li class="image-wrap">
      <a href="img/kite.jpg" title='Kite'>
        <img src='img/kite-mini.jpg'>
      </a>
    </li>
    <li class="image-wrap">
      <a href="img/tree.jpg">
        <img src='img/tree-mini.jpg'>
      </a>
    </li>
  </ul>

Important: Put src of big image (that will be shown in the modal window) into href attribute of a link in preview:

  
  <a href="big image scr here">

Optional: Set the title attribute if you want to show a caption:

  
  <a href="big image scr here" title='caption here'>

5. Include plugin call into your javascript file. Be sure that this call goes within DOMContentLoaded function:

  
  $(function() {
    $('.image-wrap a').imageBox();
  });
